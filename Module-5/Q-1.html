Module-5 jQuery Basic, Effects & Advanced

1)What is jQuery?

->It is an fast and small Javascript library.

->The purpose of jQuery is to make it much easier to use Javascript on our website.

-> jQuery is widely famous with "Write less, do more".

->jQuery contains features like;
  • HTML/DOM Manipulation
  • CSS Manipulation
  • HTML event methods
  • Effects and Animations
  • AJAX
    • ->There are several ways to start using jQuery:- 1)Using CDN(content delivery network) to include the version of jQuery.

    2)Download version of jQuery from jQuery.com and host it into your own server/local filesystem.

    ->Syn:- $(selector).action()

    ->Here, '$' sign represents jQuery.

    ->In (selector) used to find HTML elements in html page.

    ->A jQuery action() is the action to be performed on the selected elements.
