6)What is the difference between Bootstrap 4 and Bootstrap 5?

Basis Bootstrap 4 Bootstrap 5
Grid System It contains five tiers i.e.xs, sm, md, lg, xl. It contains six tiers i.e.xs, sm, md, lg, xl, xxl.
Colors It contains limited Colors. Extra colors are added and also various shades are available.
Internet Explorer It supports both IE 10 and 11. It dosen't supports IE 10 and 11.
Form Elements Radio buttons and Checkboxes have different look on different OS and browsers. Radio buttons and Checkboxes don't have different look on different OS and browsers.
The form uses whatever default browsers is provide. The forms can be customized and form controls are added, they would not depend on browsers.
Bootstrap Icons It dosen't contain own SVG icons, they have to use font awesome icons. It contains own SVG icons.
RTL Support It dosen't enable RTL(Right to Left) switching. It enables RTL(Right to left) switching.
Jumbotron It supports Jumbotron. It dosen't supports Jumbotron.
Card Deck Card deck is used to create a set of cards with equal width and height. Card deck is removed.
