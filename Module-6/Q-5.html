Grid System in Bootstrap

  • Bootstrap grid system uses a containers, rows, columns to layout and align the content.
  • It is built with flexbox and it is fully responsive.
  • It contains twelve(12) column systems in one row.
    • Six default grid tiers like:
  • Extra Small(xs)
  • Small(sm)
  • Medium(md)
  • Large(lg)
  • Extra Large(xl)
  • Extra Extra Large(xxl)
