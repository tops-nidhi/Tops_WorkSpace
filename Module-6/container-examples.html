Syntax:-
1)Default-Container(container):-It provides a responsive fixed-width container.
Syntax:-
<div class="container">
content
</div>
2)Responsive-Container:-They are responsive in nature. It includes class like container with sm,md,lg,xl,xxl
Syntax:-
<div class="container-md">
content
</div>
3)Fluid-Container:-It is used for full-width container.
Syntax:-
<div class="fluid-container">
content
</div>
