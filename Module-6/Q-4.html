4)What are the default text settings Bootstrap?

*The default text settings in Bootstrap are as follows:-

1)Font-family: Default font-family is “Helvetica Neue”, Helvetica, Arial, sans-serif.

2)Font-size: Default font-size is 16px.

3)Font-weight: Default font-weight is bold.

4)Line-height: Default line-height is 1.5.

5)Text-color: Default text-color is black(#000).

6)Text-alignment: Default text-alignment is left.

7)Text-decoration: Default text-decoration is none.

8)Text-transform: Default text-transform is none.
